# ArkeoMag Pro NoLogin v3

Temiz, kullanıcı girişi olmayan Android proje sürümü.

v3 ekleri:
- Eski APK tarzına yakın kartlı ana menü
- Alt gezinme sekmeleri
- Dedektör ekranı: kolon grafik + çizgi grafik + 3D/sonar yüzey
- Live Scan ekranı
- Area Scan: grid/ısı haritası
- Sonar: radar sweep + hedef noktaları
- CSV nokta kaydı
- Bluetooth eşleşmiş cihaz listesi
- Status bar taşması düzeltildi

GitHub Actions ile build:
Actions → Android CI → Artifacts → ArkeoMag-debug-apk
